# Morfologia
O presente código realizado em python, com o auxílio do google colab, tem por objetivo demonstrar a utilização de algumas das operações morfológicas mais importantes quando se trabalha no contexto de processamento de imagens. O código foi desenvolvido pelo professor Felipe Viel, que ministra suas aulas na UNIVALI para os alunos de Introdução à Ciência da Computação do primeiro semestre, e foi requisitado que os alunos comentassem o código de forma a entender como funciona o código e ter um primeiro contato com o Python.

O código utiliza 3 imagens para o processamento no geral, todas em preto e branco para facilitar o uso das operações:
- letra j minúscula em cursivo, limpa; <br> ![j](https://github.com/Pytlovan/Morfologia/assets/111884654/70adea19-45fe-481e-b618-0e53bd9f14c2)
- letra j minúscula em cursivo, com ruídos; <br> ![j_ruido](https://github.com/Pytlovan/Morfologia/assets/111884654/bda49bb9-47f5-4d03-862f-c73cd63312fa)
- letra j minúscula em cursivo, com furos; <br> ![j_furos](https://github.com/Pytlovan/Morfologia/assets/111884654/5c3da955-b295-45ce-826c-c669733c4dbe)
